常用Cron表达式示例
Cron表达式 说明
0/1 * * * * ? 每秒执行一次
0/5 * * * * ? 每5秒执行一次
0 * * * * ? 每分钟执行一次
0 */5 * * * ? 每5分钟执行一次
0 0 * * * ? 每小时执行一次
0 0 12 * * ? 每天中午12点执行
0 0 0 * * ? 每天凌晨0点执行
0 0 0 ? * MON 每周一凌晨0点执行
0 0 0 1 * ? 每月1号凌晨0点执行
