MapStruct 数据映射演示

映射说明

本页面展示了 UserEntity 和 UserDTO 之间通过 MapStruct 进行的自动映射结果。

  • 字段名称映射：phone → phoneNumber
  • 自定义转换：birthDate → 自动计算并映射为 age
  • 状态映射：active (boolean) → status (字符串)
  • 忽略字段：password 字段在反向映射时被忽略

映射效果演示

ID 用户名 邮箱 电话号码 出生日期 年龄 状态

MapStruct 优势

  • 编译时类型检查：如果字段类型不匹配，编译时就会报错
  • 高性能：生成的代码使用直接的 getter/setter 调用，没有反射开销
  • 减少样板代码：避免手动编写大量的映射代码
  • 易于维护：映射关系清晰，修改方便
