MinIO 文件操作指南
文件上传

可以选择上传到默认存储桶或指定存储桶，支持自定义对象名称。

文件列表

可以查看默认存储桶或指定存储桶中的所有文件，支持下载、删除、查看信息和生成分享链接。

存储桶管理

可以创建新的存储桶或检查存储桶是否存在。

注意事项
  • • 请确保 MinIO 服务器已正确配置并运行
  • • 上传的文件大小受服务器配置限制
  • • 分享链接有过期时间限制
  • • 默认存储桶名称为配置文件中指定的名称
