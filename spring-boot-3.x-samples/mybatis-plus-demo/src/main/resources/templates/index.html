Spring Boot 3.x + MyBatis Plus 示例

这个示例展示了如何在Spring Boot 3.x应用中集成和使用MyBatis Plus进行数据库操作。

示例包含完整的CRUD操作、分页查询、多表关联查询等MyBatis Plus核心功能。

功能演示

示例实现了完整的用户增删改查功能，包括：

  • 查询所有用户
  • 根据ID查询用户
  • 创建新用户
  • 更新用户信息
  • 删除用户

所有操作都可以通过API文档页面进行测试。

示例实现了MyBatis Plus的高级查询功能，包括：

  • 分页查询用户
  • 根据角色ID查询用户
  • 多表关联查询（用户和角色）

示例使用Spring的声明式事务管理，确保数据操作的一致性：

  • 创建用户并分配角色（事务保证）
  • 更新用户角色（事务保证）
