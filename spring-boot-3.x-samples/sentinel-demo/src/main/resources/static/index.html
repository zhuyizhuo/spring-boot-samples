使用说明

  1. 确保Sentinel控制台已启动（可选但推荐）：java -jar sentinel-dashboard-1.8.6.jar
  2. 访问Sentinel控制台：http://localhost:8080（默认账号密码：sentinel/sentinel）
  3. 点击上方按钮测试各种限流场景
  4. 在控制台中可以实时查看监控数据和配置限流规则
  5. 建议使用快速测试功能来快速触发限流效果
